# Запуск

1. Клонируйте репозиторий
2. Установите зависимости командой ```pip install -r requirements.txt```
3. Пример запуска: ```python main.py --files tests/fixtures/employees1.csv --report performance```
4. Запуск тестов: ```pytest```
